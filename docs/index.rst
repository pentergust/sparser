Добро пожаловать в документацию SParser!
========================================

.. image:: _images/telegram.png

Проект позволяет быстро и удобно работать с расписанием уроков.
Дополняет стандартный функционал расписания.
Предоставляет удобный обёртки, например Telegram бота.

**И того мы получаем**:

- **Автономный парсер** расписания уроков.
- Возможность **быстро и удобно** просматривать расписание.
- **Инструменты** для работы с расписанием.
- Telegram и vk чат-боты.
- Отслеживание изменений по всему расписанию.
- Автоматические **оповещения**.
- И многое другое...

Проект активно развивается и дополняется новыми возможностями.
