User storage
============

Данный раздел посвящён пользовательским храналищам.
Пользовательское хранилище отвечает за хранение и работу с
пользователя расписания.
Благодаря пользовательским хранилищам пользователя могут сохранять
свои намерения, получить изменения в расписании и многое другое.
Пользовательких хранилища используются внутри платформы.
