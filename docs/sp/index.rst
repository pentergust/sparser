Ядро SParser
============

Этот раздел посвящён ядру проекта - `SParser`.
Здесь описывается принцип работы внутренних компонентов.
Начиная с класса Schedule, которые напрямую работает с расписанием.
Генератором сообщений, для более удобной работы в среде чат-ботов.
Заканчивая счётчиками для подсчёта элементов расписания, храниилщем
пользователей и их намерений.

.. hint:: Класс пользователя

    С обновлением sp v6 произошли некоторые значимые обновления для
    генератора сообщений, касательно работы с пользователями.
    Обратите внимание на новые классы для работы с пользователем.
