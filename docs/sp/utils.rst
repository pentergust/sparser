Работа с файлами
----------------

Являются обёртками над ``ujson`` для более удобного хранения данных.
json файлы используются для сохранения и кеширование данных.

.. note:: Форматы хранилищ

    Помимо JSON в проекте также испольщуются базы данных.
    Постепенно все компоненты будут переведены на базы данных.
    Это нужно для большей производительности и удобства работы.
    Постепенно все хранилиша будут переведены на базы данных.
    Следовательно, данные фцнкции для рабоыт с json также будут
    удалены.

    Кто продолжает использовать json:
    - Расписание.
    - Список изменений
    - Индексы.
    - Храниише пользователей.

Работа с обновлениями
---------------------

.. note:: Хранилище обновлений

    В обновлении sp v7 появится храниилще изменений расписания.
    Оно будет предоставлять методы для сравнения расписаний, а также
    методы для управления записями об изменениях.
    В том числе это касается метода для совмещения записей.
